Library Management System

This project represents a library management system designed to manage books, members, and library transactions efficiently. It helps in maintaining records in an organized and systematic manner.

Book Management

The system stores details of books including title, author, publication, category, and availability status.

Member Management

The library maintains member records such as student details, membership ID, and contact information.

Issue and Return System

Members can borrow books for a specific period. The system records issue and return dates and updates availability automatically.

Fine Calculation

If a book is returned after the due date, the system calculates a fine based on the number of delayed days.

Search Facility

The system allows users to search for books by title, author, or category to quickly find required resources.
